# Portfolio Website (Draft)

This project is a draft version of my personal portfolio website, built with HTML, CSS, and JavaScript.

The goal of this project was to create an initial portfolio design that showcases my projects, skills, and contact information while practicing responsive layouts, styling, and basic JavaScript interactivity.

## Current Features

- Responsive portfolio layout
- About Me section
- Project showcase cards
- Expandable project details using JavaScript
- Contact section with social and professional links
- Custom black and neon-blue color theme

## Future Plans

This portfolio serves as a foundation for a future rebuild using React and additional modern front-end technologies.

Planned improvements include:

- React component architecture
- Enhanced user interactivity
- Improved project filtering and navigation
- Additional animations and transitions
- Backend integration for contact functionality
- Expanded project content and case studies

## Technologies Used

- HTML5
- CSS3
- JavaScript
